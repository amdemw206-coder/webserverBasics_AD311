## Project Title: AD 311 Building a Simple Web Server in Node.js
#### By: Amde Wubshet

### Breif Description: 
Creating a simple web server using Node.js. This server will serve a basic HTML page and handle different routes including a homepage and an about page. 

### Commands: 
- *node simplewebApp.js*: runs the server from the terminal. From there you can use any web browser to access the HTML page.
